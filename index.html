Top 50 HTML Tags and their uses

  • <html>:

    Defines the root of an HTML document

  • <body>:

    Contains the visible content of the document.

  • <title>:

    Defines the title of the document, shown in the browser's title bar.

  • <head>:

    Contains metadata about the document.

  • <h1>-<h6>:

    Defines headings of different levels

  • <p>:

    Defines a paragraph

  • <a>:

    Defines a hyperlink

  • <img>:

    Embeds an image

  • <ul>:

    Defines an unordered list

  • <ol>:

    Defines an ordered list

  • <li>:

    Defines a list item within <ul> or <ol>

  • <div>:

    Defines a container that can hold other elements

  • <span>:

    Defines an inline content which can be styled separately

  • <strong>:

    Defines important text, typically displayed in bold.

  • <em>:

    Defines emphasized text, typically displayed in italics.

  • <br>:

    Inserts a single line break.

  • <hr>:

    Defines a thematic break or horizontal rule.

  • <table>:

    Defines a table.

  • <tr>:

    Defines a table row.

  • <td>:

    Defines a table cell.

  • <th>:

    Defines a table header cell.

  • <form>:

    Defines a HTML form for text and data input from users.

  • <input>:

    Defines an input control.

  • <textarea>:

    Defines a multiline text input control.

  • <button>:

    Defines a clickable button.

  • <select>:

    Defines a dropdown list.

  • <option>:

    Defines an option in a <select> dropdown.

  • <label>:

    Defines a label or title for an input element.

  • <iframe>:

    Embeds another HTML page into the current document.

  • <audio>:

    Embeds audio content.

  • <video>:

    Embeds video content.

  • <source>:

    Defines multiple media resources for audios or videos.

  • <canvas>:

    Defines an area for graphics, drawn with Javascript.

  • <script>:

    Embeds Javascript codes in the document.

  • <style>:

    Defines the styling for the HTML document.

  • <meta>:

    Defines metadata about the HTML document.

  • <link>:

    Links external resources, such as stylesheets.

  • <base>:

    Specifies the base URL/target for all relative URLs in a document.

  • <abbr>:

    Defines an abbreviation or acronym.

  • <address>:

    Defines contact information for the author/owner of a document/article.

  • <cite>:

    Defines the title of a work(e.g a book, a movie etc.).

  • <code>:

    Defines a piece of computer code.

  • <pre>:

    Defines preformatted text, preserving spaces and line breaks.

  • <blockquote>:

    Defines a block of quoted texts.

  • <sub>:

    Defines subscript text.

  • <sup>:

    Defines superscript text.

  • <kbd>:

    Defines keboard input.

  • <var>:

    Defines a variable.

  • <dfn>:

    Defines a definition term.

  • <time>:

    Defines a specific period in time or a date.
